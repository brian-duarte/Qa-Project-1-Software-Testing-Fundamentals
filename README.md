# Qa-Project-1-Software-Testing-Fundamentals

## 📋 Project Description

Performed **Regression Testing** on the "Urban Routes" web application. 
Urban Routes is a logistics platform designed to create optimized routes, calculating travel duration and costs across various transportation modes

## 🎯 Objectives
* ✅**Execute** comprehensive test cases to ensure functional stability.
* ✅**Document** and track software defects.
* ✅**Generate** detailed bug reports.

## 📊 Results
* Total Test Cases Executed: 24
* Passed Cases: 17
* Failed Cases (Bugs Found): 7
* Critical/High Severity Defects: 4
